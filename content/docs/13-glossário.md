## **13. Glossário**

| Termo                  | Definição |
|:-----------------------|:----------|
| **Anel (Ring)**        | Sequência fechada de coordenadas que define o contorno de um polígono. |
| **BOM**                | Byte Order Mark - marcador de ordem de bytes em arquivos de texto. |
| **CRS**                | Coordinate Reference System - Sistema de Referência de Coordenadas. |
| **Datum**              | Superfície de referência para medições geodésicas. |
| **EPSG**               | European Petroleum Survey Group - organização que mantém um banco de dados de CRS. |
| **Feature**            | Entidade geográfica com geometria e atributos. |
| **FeatureCollection**  | Coleção de `Feature`s, o objeto raiz de um arquivo GeoJSON. |
| **Geometria**          | Representação matemática da forma e localização de um objeto espacial. |
| **GeoJSON**            | Formato de dados geoespaciais baseado em JSON. |
| **ISO 8601**           | Padrão internacional para representação de datas e horários. |
| **JSON**               | JavaScript Object Notation - formato de intercâmbio de dados. |
| **JSON Schema**        | Vocabulário para validação da estrutura de documentos JSON. |
| **LineString**         | Tipo de geometria que representa uma linha. |
| **MultiPoint**         | Tipo de geometria que representa múltiplos pontos. |
| **Point**              | Tipo de geometria que representa um único ponto. |
| **Polygon**            | Tipo de geometria que representa uma área fechada. |
| **Propriedades**       | Atributos alfanuméricos associados a uma `Feature`. |
| **RFC**                | Request for Comments - documento técnico publicado pela IETF. |
| **SIRGAS**             | Sistema de Referência Geocêntrico para as Américas. |
| **UTF-8**              | Codificação de caracteres Unicode. |
| **WKT**                | Well-Known Text - formato texto para representação de geometrias. |
